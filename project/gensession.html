Gensession - A flexible zoomable interface for melody generation

GenSession is a pioneer interface that propose an original way to interact with random music generator.

  • Gensession web page

What’s GenSession?

The system allow the user to generate melodies, organize them, manage generation parameters and construct a midi music sequence without music knowledge.

You can try it! No inspiration is required ;) And luckily there is a web page that explains all about it!

The projet was done for my master thesis in the lab of Michael McGuffin. The video that follow show how it works.

You can also download my master thesis (written in French) at http://espace.etsmtl.ca/1278/.
